# Grey Red pictures

Drop Project Grey Red artwork, renders, and progress images in this folder so they can sync into the smaller gallery on the Project Grey Red page.

Use these optional names for automatic pickup:

- `grey-red-image1.jpg` / `.png` / `.webp`
- `grey-red-image2.jpg` / `.png` / `.webp`
- `grey-red-image3.jpg` / `.png` / `.webp`

No binary image files are committed here by this change.
