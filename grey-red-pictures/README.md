# Grey Red pictures

Drop Project Grey Red artwork, renders, and progress images in this folder so they can be synced with the site later.

The small gallery on `index.html` auto-checks for these optional names and shows the ones that exist:

- `grey-red-image1.jpg`, `grey-red-image2.jpg`, `grey-red-image3.jpg`
- `grey-red-image1.png`, `grey-red-image2.png`, `grey-red-image3.png`
- `grey-red-image1.webp`, `grey-red-image2.webp`, `grey-red-image3.webp`

No binary image files are committed yet; the site uses CSS placeholder slots until final assets are ready.
