# Grey Red pictures

Drop Project Grey Red artwork, renders, and progress images in this folder so they can be synced with the site.

The album page will automatically try to show a small gallery using these filenames:

- `grey-red-image1.jpg`
- `grey-red-image2.jpg`
- `grey-red-image3.jpg`
- `grey-red-image4.jpg`
- `grey-red-image5.jpg`
- `grey-red-image6.jpg`

No binary image files are committed by this change.
